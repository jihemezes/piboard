Desktop application Opens the dashboard automatically when you log in to Windows. This section only appears in the desktop application.
Touchscreen Follows the language above (French AZERTY / English QWERTY). Applies everywhere text can be typed — tile settings and the notes widget included. Enlarges the toolbar buttons, the drawer and toolbar pull tabs, the tile icons and the resize handles, for fingers on a touchscreen.
Screen saver During the time slots below, the screen goes fully black (or shows a photo slideshow), to save energy at night or when no one is around. A click, a tap, or any key press wakes it up instantly. This never turns the physical display off — a software overlay only, which is instant and always reliable to wake up.
If the screen is woken up with a touch/click/key while still inside a time slot above, it goes back to sleep on its own after this long without any further action.
